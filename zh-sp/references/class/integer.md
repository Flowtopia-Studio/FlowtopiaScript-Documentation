# [参考](../references.md)：[类](../class.md) > 整数

> [*回到首页*](https://github.com/Flowtopia-Studio/FlowtopiaScript-Documentation)

## 介绍
整数是一个最小单位为 1 的数字。

## 例子
以下的内容都是一个整数。
```
233
```
```
0
```
```
-233
```

## 类型转换
整数可以直接被：
 - 转换为[字符串](string.md)
 - 转换为[浮点数](float.md)
 - 转换为[布尔值](bool.md)
 - 转换为[路径](path.md)
